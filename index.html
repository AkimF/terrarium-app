Terrarium

Live QML Editor and Viewer

Terrarium is an open source QML Playgrounds. The view renders lively as you type in the editor, makes prototyping and experimenting with QtQuick a lot more fun!

I'm adding this to my list of ways to show off app development.

Michael Hall | Canonical Ltd.

Install

Download it for Mac OSX, or Ubuntu Linux (amd64 / i386)

Build from Source

git clone https://github.com/penk/terrarium-app.git
cd terrarium-app && git submodule init && git submodule update 
qmake && make

About

Technically, Terrarium monitors changes in its TextEdit, and triggers the view to reload source from the local http server. If you're looking for a file system watcher implementation, please refer to QML LiveReload.
